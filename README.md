This repository contains the source code of the YOMI Hustle mod "Yharim, The God Seeker".

The mod can be found [here](https://steamcommunity.com/sharedfiles/filedetails/?id=3417828396) on the Steam Workshop.

It adds one character: Yharim, based on Godseeker Yharim from the [Calamity Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=2824688072).

Calamity's official website can be found [here](https://calamitymod.com).
